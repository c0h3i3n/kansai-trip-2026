  • 晚餐不排預約餐廳，保留彈性。

必帶 / 必查

  • Webket 確認信、4 張 QR code 截圖 / 離線票券。
  • 薄外套，館內冷氣可能較強。
  • 孩子如果會暈車，搭車前不要吃太多。
7/24 週五 · Kids Plaza + Pokémon / 梅田

目標：大阪最後彈性日，讓小孩有放電空間，大人不要排滿

保守時間表

10:00前往 Kids Plaza Osaka
10:30Kids Plaza 主活動，讓小孩充分玩
12:30午餐

晚上大阪最後晚餐，避免太晚睡

注意事項

  • Kids Plaza Osaka 通常 09:30–17:00，最後入館 16:15；出發前再看官方開館日曆。
  • Kids Plaza 是主菜，Pokémon / 梅田是加分，不是必跑。

刪減順序

  • 再刪 Pokémon Center。
  • 保留 Kids Plaza + 早回飯店整理行李。

必帶 / 必查

  • 孩子備用衣物與水壺。

7/25 週六 · 大阪退房 → 京都入住 + 清水寺 / 祇園

目標：搬飯店不要狼狽，清水寺只當京都開場，不硬排太滿

保守時間表

08:30起床、早餐、最後整理行李
10:00ESLEAD HOTEL 退房
10:30計程車到 JR 大阪站，搭 JR 京都線新快速前往京都站
12:00抵達京都站附近，午餐
12:30在 JR 綠色窗口 / 綠色售票機領取已購買的 7/29 HARUKA 5 實體票
13:30到 RESI STAY HEART 依入住說明寄放行李 / 辦理入住流程；房間尚未可進就先寄放行李
15:00搭計程車 / Uber Taxi 前往清水寺周邊

注意事項

  • 週六京都熱門區會擠，清水寺交通建議直接叫車。
  • 搬飯店日最怕行李卡住，景點要少，心情才會穩。
  • 抵達京都站時依 JR-WEST 訂位確認信先領取 7/29 HARUKA 5 實體票，避免回程日早上臨時處理。
  • RESI STAY 若房間尚未可進，先依入住說明寄放行李，再去清水寺。
  • 若入住時間受限，先寄行李，晚上再完整整理房間。

刪減順序

  • 先刪祇園深度散步。
  • 再刪二寧坂購物。
  • 最少保留：京都入住 + 清水寺短停 + 晚餐。

必帶 / 必查

  • 兩間飯店地址、京都住宿入住方式、住宿稅現金。
  • RESI STAY 行李寄放方式、入住信件與緊急聯絡資訊。
  • HARUKA 預訂資料、付款實體信用卡、4 位數認證碼、4 人護照與京都站領票位置；領票後核對日期、HARUKA 5、車廂與座位是否和離線票券一致。
  • 行李輪子狀況、雨具、孩子替換衣物。
  • 到京都後先確認回飯店路線，避免晚上迷路。
7/26 週日 · 伏見稻荷 + 錦市場 + 下午休息

7/27 週一 · 嵯峨野小火車 + 嵐山

目標：不要走反方向；先到龜岡端搭小火車，再回嵐山散步

保守時間表

必帶 / 必查

  • 前一天晚上確認 E-ticket 連結可開啟；QR code 會在乘車當日出現。
