環球影城購票重點

主軸：哈利波特 + 侏羅紀
票種：1.5 Day Studio Pass + 第 2 天 Express Pass

嵐山小火車

嵯峨野觀光小火車

預計搭乘：2026/7/27
開賣：日本時間乘車日前 1 個月 00:00
台灣時間：2026/6/26 23:00
備案：若沒買到，就改猴子公園
官方頁：嵯峨野小火車官方預約頁
平台：Klook 頁面
小火車 / 猴子公園二選一
景點購票方式備註
USJ已購買出發前確認 QR code / App / 訂單截圖
嵐山小火車建議提前搶票熱門時段容易滿
大阪城可現場買親子可安排半天內
海遊館現場或線上皆可旺日先買更安心
Kids Plaza可現場買屬一般入館型
清水寺 / 金閣寺現場買不用先搶票
伏見稻荷 / 祇園 / 錦市場免門票直接去即可
京都鐵道博物館官方票券頁 / KKday 頁面親子很適合；模擬駕駛體驗請看 體驗頁

交通備註

日期交通重點建議
7/20KIX → 大阪飯店 → USJ到飯店放行李後，下午再進 USJ
7/25大阪 → 京都 → 清水寺搬飯店日，怕公車擠可直接搭計程車 / Uber Taxi 去清水寺
7/26京都站附近 → 伏見稻荷以 JR 為主，不建議公車
7/27京都 → 嵐山搭 JR / 私鐵即可，不用特地擠公車
7/28RESI STAY HEART → 金閣寺建議直接叫車，避開京都熱門公車人潮
7/29京都 → 關西機場早上直接出發，保守抓寬鬆時間
京都公車
結論：能避就避，特別是清水寺、金閣寺兩天
比較適合叫車的日子：7/25 清水寺、7/28 金閣寺
比較適合 JR / 私鐵的日子：伏見稻荷、嵐山

備案與刪減順序

太累時
