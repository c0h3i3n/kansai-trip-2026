每日詳細攻略

7/20 週一 · 抵達大阪 + USJ 半天

目標：順利入境、放好行李、輕鬆進 USJ 熟悉環境

保守時間表

03:50已預約 Uber，從林口星空樹出發前往桃園機場 T1（NT$796）
04:20預估抵達桃園機場 T1，辦理報到 / 托運
06:40TPE 起飛
10:25抵達 KIX T1，入境、領行李、上廁所
12:00離開機場，往大阪市區 / 飯店方向
13:30抵達 ESLEAD HOTEL 附近，優先使用 lobby 自助行李寄放；正式入住時間為 16:00 起，不假設可提前進房
14:00簡單午餐，補水，整理 USJ 票券與行動電源
14:30前往 USJ 大門排隊，準備 1.5 日券 15:00 入園
15:00入園 USJ，主攻哈利波特區與拍照
19:30視體力離園，回飯店休息

注意事項

  • 第一天不要把 USJ 當衝設施日，重點是熟悉園區、拍照、吃主題餐。
  • 1.5 日券是 15:00 後入園；14:30 到大門口排隊即可，不需要更早硬趕。
  • 7/20 是日本海之日國定假日、三連休最後一天，USJ 只當暖身日，不硬排大型設施。
  • 若入境或交通延誤，USJ 可以延到 16:00 後再入園，不需要硬趕。
  • ESLEAD 正式入住 16:00 起；13:30 抵達時以寄放行李為主。
  • 出發前把 USJ QR code、飯店地址、機票截圖存離線。
  • 抵達 KIX 後先確認網路能用，再移動。
  • 翔翼 eSIM 已購買，台灣先安裝，抵達日本後手動切換啟用；不要刪除 eSIM。
  • Uber 已訂 03:50，金額 NT$796；出門前確認車牌、司機、行李件數與護照都在身上。

刪減順序

  • 若小孩累：USJ 只進哈利波特區拍照 + 逛商店。
  • 若大人累：晚餐不要排餐廳，回飯店附近便利商店 / 外帶。
  • 若航班延誤很久：取消 USJ 半天，保留隔天全天主力。

必帶 / 必查

  • 護照、機票、USJ 票券、飯店確認資料。
  • 翔翼 eSIM QR code / 開通說明 / 客服資訊截圖。
  • 行動電源、兒童水壺、濕紙巾、薄外套。
  • 行李先以能快速拿出護照、票券、充電線為原則整理。
7/21 週二 · USJ 全天

目標：用 Express Pass 把重點設施穩穩玩完，不在園區熱到崩潰

保守時間表

07:00起床、吃早餐、確認票券與 App
07:45從飯店出發前往 USJ
08:30抵達園區外，依現場開園時間排隊
上午先玩非 Express 但熱門的設施 / 哈利波特補強
11:00提早午餐，避開 12:00–13:00 人潮
下午依 Express Pass 時段跑重點設施
15:00安排室內休息、補水、買紀念品
18:00晚餐或離園，依小孩體力決定是否看夜間氛圍

注意事項

  • Express Pass 指定時段是當天最高優先級，其他安排都要讓路。
  • 7 月園區很熱，補水與室內休息要當成正式行程。
  • 哈利波特與侏羅紀是主軸，其他區域當加分。
  • 能量手環、魔杖可現場買，不需要第一時間衝商店。

刪減順序

  • 先刪排隊過長、沒有 Express 的設施。
  • 再刪大人想逛的商店，保留小孩最期待的區域。
  • 若下午太累，晚餐後直接回飯店，不硬等夜間。

必帶 / 必查

  • USJ App、所有 QR code、Express Pass 截圖。
  • 帽子、防曬、涼感巾、行動電源。
  • 前一晚先截圖園區地圖與 Express Pass 時段。

7/23 週四 · 海遊館 + 天保山

目標：USJ 後的恢復日，用大型室內水族館穩定親子節奏

保守時間表

14:00摩天輪 / 商場 / 休息，視天氣決定
15:30回飯店或回難波附近輕鬆晚餐

注意事項

  • 這天是恢復體力用，不要追加遠景點。
  • 海遊館 e-ticket 已購買，入館時間 09:30-09:45；每人需出示 1 個 QR code。
  • 海遊館就是這趟的水族館主菜，這天看完即可。

刪減順序

  • 先刪摩天輪與商場，保留海遊館。

  • 晚餐不排預約餐廳，保留彈性。

  • 薄外套，館內冷氣可能較強。
  • 孩子如果會暈車，搭車前不要吃太多。

  • 水、帽子、防曬、涼感巾。
  • 伏見稻荷到錦市場的交通路線。
  • 現金零錢，錦市場小吃可能不一定都刷卡。
7/27 週一 · 嵯峨野小火車 + 嵐山

目標：不要走反方向；先到龜岡端搭小火車，再回嵐山散步

保守時間表

08:20從 RESI STAY HEART 出發前往京都站
08:45搭 JR 嵯峨野線前往馬堀站
09:30 左右抵達 JR 馬堀站，步行約 10 分鐘到トロッコ亀岡駅
10:10 前抵達小火車龜岡站，找入口、上廁所並開啟 E-ticket
10:30搭嵯峨野4號：トロッコ亀岡 → トロッコ嵯峨
10:56抵達トロッコ嵯峨，開始嵐山行程
下車後竹林、渡月橋、午餐 / 甜點
15:00視體力回京都市區，不排猴子公園

注意事項

  • 票券方向是「龜岡 → 嵯峨 / 嵐山」，所以不要先去嵐山再想搭車。
  • 乘車 QR code 僅於 7/27 當日顯示，需開啟原始 E-ticket；前一晚確認可正常登入。
  • 小火車已購買大人 2 + 兒童 2，已付款 JPY 2,640。
  • 座位為嵯峨野4號、4號車16A–16D。
  • 猴子公園預設刪除，避免爬坡透支體力。

刪減順序

  • 小火車不可刪，已購票。
  • 竹林與渡月橋保留短版即可。
  • 若天氣太熱或下雨，午餐後直接回京都。

必帶 / 必查

  • E-ticket、手機電量、離線截圖。
  • 前一天晚上確認 E-ticket 連結可開啟；QR code 會在乘車當日出現。
  • 查好 JR 京都站到馬堀 / 龜岡方向的月台。
7/28 週二 · 金閣寺 + 京都鐵道博物館

目標：上午短戶外，下午室內避暑；金閣寺 + 京都鐵道博物館為主

保守時間表

07:30起床、早餐，補水防曬
08:30從 RESI STAY HEART 叫計程車 / Uber Taxi 前往金閣寺
08:55抵達金閣寺門口，準備 09:00 開門第一波入場
09:00金閣寺，停留約 45–60 分鐘
10:00離開金閣寺，前往梅小路 / 京都鐵道博物館方向
10:45抵達梅小路 / 京都鐵道博物館周邊
11:00京都鐵道博物館，先確認 SL / 體驗項目時間
12:30館內或周邊午餐
15:30視體力離開，回飯店整理行李
晚上早點休息，準備隔天回程

注意事項

  • 金閣寺是戶外景點，7 月底不要停太久。
  • 金閣寺 09:00 開門，早到可以避開 10:00 後的烈日與旅行團人潮。
  • 鐵道博物館和海遊館較不重複，且室內避暑能力好。
  • 京都鐵道博物館通常 10:00–17:00，最後入館 16:30；想玩體驗項目就不要太晚進館。
  • 晚上要整理回程行李，不要排晚間景點。

刪減順序

  • 若太熱：金閣寺縮短到拍照 + 繞一圈。
  • 若小孩不想看鐵道：縮短博物館，改梅小路公園或京都站附近休息。
  • 若大人小孩都累：金閣寺後直接回飯店休息。

必帶 / 必查

  • 京都鐵道博物館營業日、票券方式、SL / 體驗項目時間。
  • 回程行李清單，晚上打包不要漏。
  • 隔天 KIX 交通路線與 HARUKA 5 實體票；確認已於 Day 5 在京都站完成領票。
7/29 週三 · 京都退房 → KIX → 回台

目標：不 delay；寧可早到機場，不賭交通

保守時間表

05:30起床，早餐以便利商店 / 車站簡單解決
06:00完成退房，確認護照 / 機票 / 行李
06:10離開 RESI STAY HEART，搭計程車前往京都站
06:25–06:35抵達京都站，依當日看板前往 HARUKA 月台
06:44搭乘 HARUKA 5；普通車指定席 4號車6A–6D
08:19抵達關西機場站，前往 KIX T1 辦理 check-in / 托運行李
10:30前到登機門
11:15KIX 起飛回台

注意事項

  • 回程日不加任何景點，不買早餐名店，不排伴手禮店。
  • 前一晚把護照、錢包、手機、票券、行動電源放在同一個隨身包。
  • 如果早上叫車不順，立刻改步行 / 地鐵到京都站，不要等太久。
  • Day 5 到京都站時先領取 7/29 HARUKA 5 實體票，回程日直接進站。
  • 目標 8:30 前到 KIX T1，這樣就算排隊也還有緩衝。

刪減順序

  • 刪早餐餐廳。
  • 刪京都站伴手禮。
  • 刪機場慢慢逛，先完成 check-in、托運、安檢。

必帶 / 必查

  • 護照、機票、住宿退房確認、交通 IC 卡 / 現金。
  • 前一晚確認 HARUKA 5 實體票、06:44 班次與當日月台資訊。
  • 行李秤重，托運 20kg 與隨身限制不要超。
